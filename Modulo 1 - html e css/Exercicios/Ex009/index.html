Tipos de listas:

Listas ordenadas

Para criar uma lista ordenada utilizo a tag "ol"
Exemplo:
  1. item 1
  2. item 2
  3. item 3

Também posso modificar o marcador para letras ou números romanos, o padrão são números

Exemplo:
  1. item a
  2. item b
  3. item c

observação:

quando a lista for ordenada baseada em letras, mas sendo necessário começar por determinada letras deve colocar o numero dessa letra para o poder o parâmetro start entender.
Exemplo:
  1. item 1
  2. item 2
  3. item 3

Listas desordenadas

Para uma lista Desordenada utilizo a tag "ul"
Exemplo:
  • item 1
  • item 2
  • item 3

Também posso modificar o marcador para quadrado ou circulo, o padrão é uma bola

Exemplo:
  • item 1
  • item 2
  • item 3

Listas misturadas

Posso misturar as tags e criar uma lista mista
Exemplo:
  1. item 1
  2. item 2
    • item alternativo
    • item alternativo
    1. item a
    2. item b
Exemplo com comidas:
  • café da manhã
    1. pão
      • com queijo
      • com ovo
    2. café
      • com leite
      • puro
  • almoço
    1. lasanha
      • queijo e presunto
      • ou com carne
    2. galeto
      • completo
      • bem assado
